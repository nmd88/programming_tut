jQuery Basics

jQuery is mainly used for :



  • Finding elements in an HTML document
  • Changing HTML content
  • Listening for user interaction and changing behaviour depending on the input
  • Animating the content on the page
  • Sending requests to servers and retrieving new content

Document Object Model



A tree-like structure created by browsers so we can quickly find HTML elements using javascript

Inside the DOM, HTML elements become nodes which have relationships with one another

There are two types of nodes: element and text

Javascript is the programming language that allows developers to interact with the DOM

You can use jQuery to interact with the DOM and it will work on most modern browsers

jQuery usage



This is jQuery syntax: jQuery(-selector-)

This is jQuery simplified syntax: $(-selector-)

Generally, you would use the CSS selectors (on jQuery) to find elements inside the DOM
